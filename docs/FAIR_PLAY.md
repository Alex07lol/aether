# Fair Play Policy

Aether is not a hacked client. The codebase must never include:

- KillAura, Aim Assist, TriggerBot, AutoClicker, or combat automation.
- Reach hacks, velocity modification, anti-knockback, fly, speed, scaffold, blink, or no-slow.
- X-Ray, ESP, hidden-player revealers, or other unfair information overlays.
- Packet manipulation intended to bypass server rules or create an advantage.

Allowed features should be quality-of-life, visual, informational, cosmetic, or accessibility-oriented. Some servers still restrict otherwise legitimate client-side features, so modules should be easy to disable per profile/server.

The `FairPlayPolicy` class is a guardrail, not the whole review process. New modules should still be reviewed for behavior, not only their name.
